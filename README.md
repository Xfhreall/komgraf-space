# Komgraf-Space

## Deskripsi

Komgraf-Space adalah proyek berbasis web yang dibangun menggunakan Vite dan JavaScript. Proyek ini berfungsi sebagai ruang eksperimen untuk pengembangan grafis komputer, visualisasi, serta percobaan antarmuka berbasis web.

## Fitur

* Struktur proyek modern menggunakan Vite.
* Mendukung modul ES dan hot-reload untuk proses pengembangan cepat.
* Aset statis terorganisasi di folder `static/`.
* Siap diperluas untuk kebutuhan visualisasi grafis atau komputasi grafis.

## Prasyarat

Pastikan Anda telah menginstal:

* **Node.js** (versi sesuai kebutuhan proyek)
* **npm**

## Instalasi & Menjalankan Proyek

1. Clone repository:

   ```bash
   git clone https://github.com/Xfhreall/komgraf-space.git
   ```
2. Masuk ke folder proyek:

   ```bash
   cd komgraf-space
   ```
3. Install dependency:

   ```bash
   npm install
   ```
4. Jalankan mode pengembangan:

   ```bash
   npm run dev
   ```

## Struktur Proyek

```
komgraf-space/
├── src/            # Kode sumber utama (JavaScript)
├── static/         # Aset statis seperti gambar/CSS
```
